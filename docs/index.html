Overview

Evaluation claims need more than a single score.

Two benchmarks can report strong performance while supporting very different claims. The survey separates the capability target, the probing protocol, and the scoring evidence so that benchmark results can be interpreted on comparable terms.

01

Evaluation Target

Specifies the property of world-state evolution or downstream use being characterized.

02

Evaluation Protocol

Distinguishes open-loop prediction, controlled rollout, and closed-loop interaction.

03

Evaluation Evidence

Clarifies whether scores rely on automatic, judgment-based, or outcome-grounded evidence.
